## 📋 About
Speech to Text Tool is a powerful desktop application that converts audio and video files into text using OpenAI's Whisper model. Features a user-friendly interface and supports multiple languages and formats.

## ✨ Key Features
- 🎙️ Advanced speech recognition
- 🎥 Video file support
- 🌍 Multi-language support
- 📝 Real-time transcription
- 📤 Multiple export formats

<div align="center">
<img src="public/features.gif" alt="Features Demo" width="600"/>
</div>

## 🛠️ Tech Stack
- 🐍 Python 3.9+
- 🤖 OpenAI Whisper
- 🖥️ PyQt6
- 🎬 FFmpeg
- 📝 Word & PDF export

## 🚀 Quick Start
```bash
# Clone repository
git clone https://github.com/Ivantech123/Speech-to-text-tool.git
cd Speech-to-text-tool

# Install dependencies
pip install -r requirements.txt

# Run the application
python main.py
```

## 📋 О проекте
Speech to Text Tool - мощное десктопное приложение для конвертации аудио и видео файлов в текст с использованием модели OpenAI Whisper. Имеет удобный интерфейс и поддерживает множество языков и форматов.

- 🌍 Поддержка разных языков
- 📝 Транскрипция в реальном времени
